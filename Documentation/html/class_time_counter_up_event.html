Opis szczegółowy

Encja reprezentująca zdarzenie odmierzania czasu w górę w aplikacji.

Klasa reprezentująca zdarzenie, od którego jest odmierzany czas. Posiada swoje unikatowe Id, nazwę zdarzenia oraz powiązany z nim zegar (TimeCounterUp).
