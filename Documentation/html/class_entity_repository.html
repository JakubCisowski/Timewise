Opis szczegółowy

Generyczna klasa implementująca interfejs IGenericRepository, będąca jego konkretną implementacją. Klasa implementuje również interfejs IDisposable, który pozwala na automatyczne zwolnienie zasobów po zakończeniu działania, oraz na tworzenie obiektu repozytorium wewnątrz bloku using.
