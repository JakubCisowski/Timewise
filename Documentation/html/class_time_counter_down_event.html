Opis szczegółowy

Encja reprezentująca zdarzenie odmierzania czasu w dół w aplikacji.

Klasa reprezentująca zdarzenie, do którego jest odliczany czas. Posiada swoje unikatowe Id, nazwę zdarzenia oraz powiązany z nim zegar (TimeCounterDown).
